# ライセンス

## プロジェクトの由来について

このプロジェクトは、Qiitaの記事「[クリスマスの夜に「RFC8762: 双方向遅延測定プロトコル STAMP」について勉強しよう!!](https://qiita.com/Kinukui2003/items/af9e4a3dfea71cc2c152)」に掲載されたGoogle Geminiが生成した小規模なサンプルコードをベースに、大幅に拡張・改良して作成されました。
s
### 派生元のコード

- **元のコード**: Qiita記事内のGemini生成サンプルコード
- **元のコードの特徴**:
  - RFC 8762の基本的な概念を示す教育的なサンプル
  - 最小限の実装

### 現在のコードの拡張内容

元のコードを以下のように大幅に拡張しました：

1. **クロスプラットフォーム対応**
   - Windows (Winsock2)
   - Linux (POSIX)
   - macOS (POSIX)

2. **エラーハンドリングの強化**
   - システムコールの戻り値チェック
   - 適切なエラーメッセージ
   - グレースフルなシャットダウン

3. **機能の追加**
   - 詳細な統計情報
   - タイムアウト処理
   - シグナルハンドリング

4. **タイムスタンプ精度の向上**
   - Windows: QueryPerformanceCounter
   - UNIX: clock_gettime with CLOCK_REALTIME

5. **テストコードの追加**
   - ユニットテスト
   - ビルドシステムの整備

6. **ドキュメントの充実**
   - README
   - ユースケース
   - ロードマップ
   - 貢献ガイド

このプロジェクトは、システム標準ライブラリのみを使用しており、外部ライブラリには依存していません：

これらのライブラリは、各OSベンダーのライセンス条項に従って使用されています。
